## Setup:

```bash
git clone [https://github.com/amitsingh6391/tap_invest_assignment.git](https://github.com/amitsingh6391/tap_invest_assignment.git)
```

```bash
flutter clean
```

```bash
flutter pub get
```

```bash
dart pub run build_runner build -d
```

```bash
flutter run
```
